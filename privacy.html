Política de Privacidade — Lena (@lena.sys)

Última atualização: 28/06/2026

Quem somos

Esta página descreve como tratamos dados relacionados ao perfil Lena (@_lena.sys no Instagram, @lena.sys3 no TikTok), uma persona de conteúdo gerada por IA mantida pela nossa agência. Contato: lena.sys.studio@gmail.com.

Aviso de transparência

Lena é uma persona de conteúdo gerada por inteligência artificial, não uma pessoa real. Todo o conteúdo visual é gerado por modelos de IA. Isso é divulgado publicamente na bio dos perfis.

Que dados coletamos

  • Visitantes/seguidores das redes sociais: não coletamos, armazenamos ou processamos dados pessoais de seguidores ou de quem interage com os posts (curtidas, comentários, mensagens) além do que a própria plataforma (Instagram/TikTok) já processa sob a política de privacidade dela.
  • Aplicativo de publicação (integração via API): usamos a API oficial de cada plataforma (Instagram Graph API, TikTok Content Posting API) somente para publicar conteúdo na nossa própria conta. O aplicativo não coleta, lê nem armazena dados de terceiros — ele só envia (publica) mídia e legenda que nós mesmos produzimos.
  • Links de afiliados: posts podem conter links de afiliados (ex.: Shein, Amazon). O rastreamento de cliques/conversão é feito pela plataforma de afiliados correspondente, sob a política de privacidade dela — não temos acesso a dados pessoais de quem clica.

Com quem compartilhamos dados

Não vendemos nem compartilhamos dados pessoais com terceiros. As únicas integrações técnicas são com as próprias plataformas (Meta/Instagram, TikTok) para fins de publicação de conteúdo.

Retenção

Não mantemos banco de dados de informações pessoais de usuários/seguidores. Conteúdo publicado (imagens, legendas, métricas agregadas e anônimas de desempenho) é mantido para fins de gestão da própria conta.

Crianças

Este perfil não é direcionado a menores de 13 anos e não coleta intencionalmente dados de crianças.

Alterações

Podemos atualizar esta política. A data no topo desta página reflete a versão mais recente.
